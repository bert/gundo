void gundo_make_redo_sensitive( GtkWidget *widget, GundoSequence *seq )

Makes a widget sensitive to the current redo state of a GundoSequence

Documentation

Makes a widget sensitive to the current redo state of a GundoSequence. A widget that is redo-sensitive will only be sensitive when it is possible to call gundo_sequence_redo on its associated GundoSequence (that is, when gundo_sequence_can_redo returns TRUE).
Parameters:
widget - The widget to make redo-sensitive.
seq - The undo sequence that the widget should be sensitive to.
