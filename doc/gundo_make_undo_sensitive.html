void gundo_make_undo_sensitive( GtkWidget *widget, GundoSequence *seq )

Makes a widget sensitive to the current undo state of a GundoSequence

Documentation

Makes a widget sensitive to the current undo state of a GundoSequence. A widget that is undo-sensitive will only be sensitive when it is possible to call gundo_sequence_undo on its associated GundoSequence (that is, when gundo_sequence_can_undo returns TRUE).
Parameters:
widget - The widget to make undo-sensitive.
seq - The undo sequence that the widget should be sensitive to.
