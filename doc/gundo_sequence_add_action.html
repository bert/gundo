void gundo_sequence_add_action( GundoSequence *seq, const GundoActionType *type, gpointer data )

Adds an action to the end of the GundoSequence

Documentation

Adds an action to the end of the GundoSequence. Any actions that have been undone but not redone are destroyed. If any groups have been started, the action is appended to the end of the most recently started group.
Parameters:
seq - The undo sequence to which to add an action.
type - The type of the action.
data - Data about the action. The data is destroyed by calling the "free" callback of the action type.
